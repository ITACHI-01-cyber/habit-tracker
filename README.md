# 📊 Habit Tracker

## ✨ Features
- **📅 Calendar Grid View** - Mark "DSA Problems", "Gym", "College Study" daily
- **🎯 Real-time Progress** - 8% DSA, 4% Gym tracking with color-coded cells
- **📈 Streak Counter** - Monthly heatmaps + consistency stats
- **💾 Local Storage** - No backend, works offline forever
- **📱 Responsive Design** - Perfect on desktop, tablet, mobile
- **⚡ Instant Load** - Vanilla JS, no frameworks

## 🛠️ Tech Stack

| Component | Technology |
|-----------|------------|
| **Frontend** | HTML5, CSS3 Grid/Flexbox, Vanilla JavaScript |
| **Storage** | `localStorage` + JSON |
| **Styling** | Custom CSS (Tailwind-inspired) |
| **Date Handling** | Native JS Date API |

## 🚀 Quick Start

```bash
# Clone & run instantly
```

1. **Add Habits** - "DSA Problems", "Gym Workout", "College Study"
2. **Daily Check-ins** - Click cells ✅/❌
3. **Auto Progress** - % completion updates instantly
4. **Monthly View** - See your consistency heatmap
5. **Data Persists** - localStorage keeps everything
